Configurar projeto NodeJs + TypeScript

  1. Criar uma pasta
  2. npm init -y
  3. npm install typescript -D
  4. npm install @types/node -D
  5. npx tsc --init
    1. Abra o arquivo “tsconfig.json”
    2. Troque a opção “target” de “es5” para “es6”
    3. Descomente a linha “sourceMap:true”
  6. npm install ts-node -D
  7. npm install nodemon -D
  8. Abra o arquivo “package.json”, na seção “scripts”.
    1. "scripts": {
      "test": "echo \"Error: no test specified\" && exit 1",
      "build": "tsc",
      "build:live": "nodemon --exec ./node_modules/.bin/ts-node -- ./index.ts",
      "dev": "npm run build:live",
      "start": "node ./index.js",
      "prod": "npm run build && npm start"
      }
  9. Mongoose:
    1. npm install mongoose --save
    2. npm install @types/mongoose --save-dev
  10. Express:
    1. npm install express --save
    2. npm install @types/express --save-dev
  11. CORS:
    1. npm install cors --save
    2. npm install @types/cors --save-dev
  12. Body-parser (middleware express):
    1. npm install body-parser --save
    2. npm install @types/body-parser --save-dev
  13. Codar o projeto:
  14. Criar arquivo index.ts
  15. let saudacao: string = 'Alô, mundo!';
    console.log(saudacao);
  16. npm run dev
  17. Done!!!
